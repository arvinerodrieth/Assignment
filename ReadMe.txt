Set Up the Project Directory

Create a new folder for your project:

Name it something like url-shortener.


It allows you to shorten long URLs and redirect users to the original URL when the shortened URL is accessed.

How to Use

1. Open index.html in a web browser.
2. Enter a long URL in the input field and click the "Shorten URL" button.
3. The shortened URL will be displayed below the button.
4. To test redirection, copy the shortened URL and paste it in the browser's address bar.

Files

- index.html: The main HTML file with the input field and button.
- Script.js: JavaScript file containing the logic for shortening and decoding URLs.
